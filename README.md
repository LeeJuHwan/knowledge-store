**나의 첫 개발** 학창 시절 [Pawn-Lang](https://github.com/pawn-lang) 으로 이루어진 해외 GTA 멀티 플레이 모드를 수정 하는 것이 첫 개발의 시작이었습니다. \
\
동시 접속자 50명 게임 서버를 호스팅 하며 희열을 느꼈던 감정과 디도스와 자료 수집 중 랜섬웨어로 컴퓨터를 잃은 감정은 아직 잊지 못합니다.\
\
그 후 수 차례 랜섬웨어에 걸렸었는데 나름의 면역이 생겼었죠. 중요 파일을 항상 USB에 백업 해두고 파일이 메모장으로 잠기는 순간 전부다 날리고 USB를 불러오면 됐던터라 나름의 버저닝을 하고 있었네요.\
\
그 당시 밥도 마다하며 컴퓨터 앞에 10시간 이상 살아오던 보잘 것 없던 제가 학창시절의 저를 다시 한 번 믿기로 했습니다.\
\
**개발자** 2023.03.04 첫 회사를 입사 하고 적성에 맞다라고 판단 할 수 있는 기간이 흘렀습니다. 나름 적성에 맞는 것 같기 때문에 이젠 소프트웨어 세계를 떠돌며 여행 하는 부랑자가 되려고 합니다.
